Attributes

src="" - Source Attribute

Specifies the URL of the resource to embed.

<img src="image.jpg" alt="Example Image">
<video src="video.mp4">
<audio src="audio.mp3">

alt="" - Alternative Text Attribute

Provides alternative text for images.

<img src="image.jpg" alt="Example Image">

href="" - Hyperlink Attribute

Specifies the URL of the page the link goes to.

<a href="https://example.com">Visit Example</a>

target="" - Target Attribute

Specifies where to open the linked document.

<a href="https://example.com" target="_blank">Visit Example</a>

rel="" - Relationship Attribute

Specifies the relationship between the current document and the linked resource.

<link rel="stylesheet" href="style.css">

type="" - Type Attribute

Specifies the type of the element.

<input type="text" name="username">

for="" - For Attribute

Associates a label with an input element.

<label for="username">Username:</label>
<input type="text" id="username" name="username">

value="" - Value Attribute

Specifies the value of an input element.

<input type="text" name="username" value="JohnDoe">

placeholder="" - Placeholder Attribute

Specifies a short hint that describes the expected value of an input element.

<input type="text" name="username" placeholder="Enter your username">

action="" - Action Attribute

Specifies the action to be performed when the form is submitted.

<form action="/submit" method="post"></form>

method="" - Method Attribute

Specifies the HTTP method to use when submitting the form.

<form action="/submit" method="post"> </form>

name="" - Name Attribute

Specifies the name of an input element.

<input type="text" name="username">

id="" - ID Attribute

Specifies a unique identifier for an element.

<div id="main-content"></div>
<input id="username" name="username">

class="" - Class Attribute

Specifies one or more class names for an element.

<div class="container"></div>

required - Required Attribute

Specifies that an input field must be filled out before submitting the form.

<input type="text" name="username" required>

value="" - Value Attribute

Specifies the value of an input element.

<input type="text" name="username" value="JohnDoe">

maxlength="" & minlength="" - Max Length and Min Length Attributes

Specifies the maximum and minimum number of characters allowed in an input element.

<input type="text" name="username" maxlength="20" minlength="2">

min="" & max="" - Min and Max Attributes

Specifies the minimum and maximum values for an input element.

<input type="number" name="age" min="0" max="120">

HTML Boilerplate

The basic starting template for any HTML page.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>
